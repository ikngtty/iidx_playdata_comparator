プライバシーポリシー

1. 取得する情報

本サービスでは、ログイン機能の提供のため、X（旧Twitter）アカウントによる外部認証を利用しています。
ログインにあたり、当該サービスからユーザーIDおよび表示名等の情報を取得します。
また、ユーザーが本サービスに登録した情報を取得し、保存します。
取得した情報は、本サービスの提供および運営のために利用します。

2. 投稿情報の公開

ユーザーが本サービスに登録したデータは、原則として公開されます。
公開されることを理解した上でご利用ください。

3. 外部サービスの利用

本サービスは、Google LLCの提供するFirebaseを利用しています。
取得した情報は、国外のサーバーに保存される場合があります。
法令に基づく場合を除き、取得した情報を第三者に提供しません。

4. 情報の開示・訂正・削除

ユーザーは、本サービスが保有する自己の情報について、開示・訂正・削除を運営者へ依頼することができます。
